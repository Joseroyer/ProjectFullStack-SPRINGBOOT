Perguntas e Resposta

  • Como faço para cadastrar uma piada?

    É necessário efetuar um cadastro de usuário e, após isso, efetuar corretamente o cadastro de novas piadas.

  • Posso remover uma piada já cadastrada ?

    Sim, basta efetuar o login e utilizar o botão EXCLUIR PIADA.

  • Posso avaliar alguma piada ?

    Sim, a avaliação de cada piada é totalmente válida e é importante o seu feedback.
